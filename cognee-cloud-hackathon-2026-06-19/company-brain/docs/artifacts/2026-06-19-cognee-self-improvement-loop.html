Two Different Kinds of “Self-Improvement”

Presentation hook: Cognee improves the worker. Our lint loop improves the trusted company memory. Same word, different target.

Cognee native skill loop

Improves the worker

“Answer this kind of question better next time.”

Skill instructions
→
Feedback proposal
→
Improved skill behavior
  • Target: QA skill, rubric, answer behavior.
  • API shape: content_type="skills", skill_improvement, improve().
  • Does not, by itself, prove that bad memory was safely removed.

Demo line: Cognee can adapt the way it answers.

Our Company Brain lint loop

Improves the memory

“Which fact is allowed to be authoritative?”

Messy facts
→
Discriminator + drift debt
→
Clean memory + ledger
  • Target: contradictory, stale, duplicate, unsafe, or unsourced facts.
  • Modules: drift debter, discriminator/router, imposer, ledger.
  • Every override, hold, merge, or quarantine writes a receipt.

Demo line: wrong answer → lint → right answer, with proof.
