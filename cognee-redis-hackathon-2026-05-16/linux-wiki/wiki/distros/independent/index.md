# Independent Distros

Independent distributions are tracked here when they do not primarily derive from one of the major upstream families.

## Distros

- Alpine Linux
- Gentoo
- Void Linux
- Solus

## Shared Notes

This section should be organized carefully because independent distros often differ sharply in package management, init systems, release models, and target users.
