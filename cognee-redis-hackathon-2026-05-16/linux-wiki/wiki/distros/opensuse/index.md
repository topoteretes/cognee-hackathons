# openSUSE Family

openSUSE-family systems include Tumbleweed, Leap, Slowroll, and SUSE relatives.

## Distros

- openSUSE Tumbleweed
- openSUSE Leap
- openSUSE Slowroll

## Shared Notes

- Package tools: `zypper`, `rpm`
- Configuration tooling often includes YaST
- Release models include rolling, regular release, and slower rolling options
