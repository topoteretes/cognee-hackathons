# Debian Family

Debian-family systems include Debian, Ubuntu, Linux Mint, Pop!_OS, elementary OS, and related derivatives.

## Distros

- Debian
- [[ubuntu]] - Debian-based Canonical distribution and upstream base for many derivatives
- Linux Mint
- [[pop-os]] - Ubuntu-based System76 distribution with COSMIC Desktop
- elementary OS

## Shared Notes

- Package tools: `apt`, `dpkg`
- Common package formats: `.deb`, Flatpak, Snap depending on distro
- Release models vary between stable, interim, and long-term support releases
