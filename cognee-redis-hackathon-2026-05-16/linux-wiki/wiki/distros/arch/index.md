# Arch Family

Arch-family systems share Arch Linux as the upstream base or primary package ecosystem.

## Distros

- [[cachyos]] - Performance-focused Arch-based distribution with optimized packages and kernels
- [[omarchy]] - Opinionated Arch + Hyprland distribution by DHH

## Shared Notes

- Package manager: `pacman`
- Common community package source: AUR
- Release model: rolling release for Arch and most close derivatives
