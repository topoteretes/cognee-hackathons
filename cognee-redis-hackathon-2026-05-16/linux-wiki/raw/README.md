# Raw Sources

Store immutable source material here.

- `articles/` clipped articles and blog posts
- `docs/` official documentation snapshots or notes
- `release-notes/` distro and project release notes
- `forum-notes/` forum threads and community reports
- `videos/` transcripts or notes from video sources
- `assets/` images and attachments

The LLM may read these files but should not rewrite them.
